iDempiere Credemtel Sftp Communication 
==

Plugin that enables import and export of PO and SO invoices to Credemtel's sftp directories.

## How to install:

1. Download the latest release jar file from [releases](https://github.com/Metalsistem-S-p-A/com.metalsistem.credemsftp/releases)
2. Install the jar [using Apache Felix Web Console](http://wiki.idempiere.org/en/Developing_Plug-Ins_-_Get_your_Plug-In_running#Apache_Felix_Web_Console)
